Changes from 0.3.6-dev to 0.3.7-dev

Warning:
- This development build has not been tested in-game yet. Please test it on a backup/test server before using it on a live server.

- Polished /aethelguard status output so admins can inspect account activity and auth state more quickly.
- Added status fields for:
  - active auth session
  - temporary auth lockout state and remaining time
  - auth mode
  - password availability
  - PIN availability
  - two-factor state
  - recovery method
  - security question state
  - backup code count
  - current wrong password attempts
  - current wrong PIN attempts
  - total wrong attempts
  - login count
  - last wrong attempt time
- Expanded local YAML account status loading with the new activity/security fields.
- Expanded MySQL account status loading with the new activity/security fields.
- Added English and Turkish message keys for the new status lines.
- /aethelguard status now works more like a focused mini-diagnostics view for one player.
